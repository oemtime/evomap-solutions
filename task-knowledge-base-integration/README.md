# 外部知识库整合到 Agent 协作决策

## 项目概述

本项目实现了将外部知识库整合到 Agent 协作决策流程的完整解决方案。

## 目录结构

```
task-knowledge-base-integration/
├── docs/
│   ├── architecture.md      # 架构设计文档
│   └── performance.md       # 性能评估报告
├── src/
│   └── knowledge_integration.py  # 核心实现代码
├── examples/
│   └── agent_collaboration_demo.py  # 集成示例
├── tests/
│   └── performance_benchmark.py     # 性能测试
└── memory.md                # 任务记忆/进度追踪
```

## 核心功能

### 1. 知识库接入架构
- 支持 RAG (检索增强生成)
- 向量数据库存储
- 多源知识连接器

### 2. 多 Agent 知识共享
- 中央仓库模式
- 联邦模式
- 混合模式

### 3. 动态知识检索
- 查询重写
- 多源检索
- 结果重排序

### 4. OpenClaw 记忆集成
- 双向同步
- 冲突解决
- 统一知识管理

## 快速开始

### 安装依赖

```bash
pip install numpy
```

### 运行示例

```bash
cd examples
python agent_collaboration_demo.py
```

### 运行性能测试

```bash
cd tests
python performance_benchmark.py
```

## 核心组件

### KnowledgeCoordinator
中央知识协调器，负责 Agent 注册、知识路由和共享协调。

### RAGEngine
RAG 引擎，提供文档索引、向量检索和上下文组装功能。

### OpenClawMemoryIntegration
OpenClaw 记忆系统集成器，实现双向知识同步。
